# uv constraints for the macOS x86_64 cibuildwheel test environment.
#
# cryptography>=49 dropped macOS x86_64 wheels (48.x shipped a universal2 wheel
# covering Intel; 49.x ships arm64-only + sdist). On the arm64 CI runner the
# x86_64 test venv would otherwise source-build cryptography and fail to
# cross-compile openssl. Cap it at the last release with an Intel wheel.
#
# This is a constraint, so it caps the transitive neuroglancer -> google-auth ->
# cryptography resolve during the wheel install, not just top-level test deps.
# Applied via UV_CONSTRAINT only for the macos_x86_64 build; affects test
# resolution only, never the shipped wheel.
cryptography<49
